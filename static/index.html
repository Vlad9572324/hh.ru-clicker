🌐 Браузерные сессии
➕ Добавить сессию из браузера
Самый простой способ — Copy as cURL:
1. Откройте hh.ru, войдите в нужный аккаунт
2. F12 → вкладка Network → любой запрос к hh.ru
3. Правая кнопка на запросе → Copy → Copy as cURL
4. Вставьте сюда весь скопированный текст — бот сам вытащит cookies
Скопируйте значения ключей из браузера:
F12 → Application → Cookies → https://hh.ru
hhtoken *
_xsrf *
hhul
crypted_id
Имя (необязательно — определится автоматически)
Сопроводительное письмо (необязательно)
🔑 Обновить куки аккаунтов
Вставьте новый cURL или строку cookie: hhtoken=… — куки обновятся сразу без перезапуска.
Получить: F12 → Network → любой запрос к hh.ru → правая кнопка → Copy as cURL
🎯 Поиск и отклики
⚙️ Параметры бота
🏢 Фильтры и автоматизация
Региональный поддомен HH
по умолчанию пусто → главный hh.ru. Пример: syktyvkar, moscow, vologda. OAuth и chatik всегда на основном.
Род соискателя для LLM-ответов
Подставляется в системный промпт LLM и дефолтный ответ опросника. Кастомные шаблоны не затрагиваются.
Умный поиск
Фильтры добавляются к поисковым URL. Работают для веб-поиска и OAuth.
Формат работы
Отмечены = разрешены. Если ничего не отмечено — пропускаются все форматы (без фильтра).
🏷️ Фильтр по словам в названии
Регистронезависимо, по подстроке. Применяется до отклика — экономит запросы.
✅ Разрешить (include) — если непусто, оставляем только совпавшие
🚫 Исключить (exclude) — выкинуть если содержит
Автопрохождение тестов
Если включен LLM — вопросы заполнит нейронка. Иначе используются шаблонные ответы.
Пред-проверка вакансий
Делает дополнительный запрос к HH для каждой вакансии. Замедляет отклики, но пропускает неподходящие.
🛑 Контроль лимитов
Если включено — бот ставит паузу при лимите. Снимите паузу вручную или дождитесь 00:00 МСК.
🔑 Метод откликов
Если выключено — отклики через Web (cookies). OAuth быстрее и не требует капчи, но один запрос за раз (не пакетом).
🔗 Пул поисковых запросов
Добавьте URL-адреса поиска вакансий — они появятся как чекбоксы на карточке каждого аккаунта. Поле стр. задаёт глубину поиска (кол-во страниц) для конкретного запроса; если пусто — используется глобальное значение из слайдера выше.
✨ Подсказки по резюме
Анализирует title/роль/скилы резюме и предлагает поисковые запросы с цифрами конкуренции (как в Аудите). Кликни ➕ — добавлю в пул выше.
📋 Шаблоны
✉️ Шаблоны писем
Создайте именованные шаблоны — они появятся в выпадающем списке на каждой карточке аккаунта.
📝 Шаблонные ответы на опросы
Когда вакансия требует опрос — бот автоматически заполнит его. Для каждого шаблона укажи ключевые слова (через запятую) и ответ. Первый совпавший шаблон используется. Последний — ответ по умолчанию.
🚀 Загрузить готовые пресеты
⚠️ Заменит текущие шаблоны. Отредактируй ответы под себя и нажми «Сохранить».
🤖 AI / LLM
🤖 LLM Авто-ответы
Бот будет автоматически отвечать на сообщения работодателей в переговорах со статусом Интервью. Используется OpenAI-совместимый API (GPT-4o-mini, DeepSeek и др.)
💡 Включить/выключить авто-ответы — кнопка во вкладке 🤖 LLM Ответы
📝 Есть несохранённые черновики — включи «Автоотправка» и они уйдут на следующем цикле без новых LLM-вызовов.
Если включено — LLM сам отвечает на вопросы работодателя при отклике. Шаблоны используются как запасной вариант.
